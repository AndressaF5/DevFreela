# DevFreela

Projeto de desenvolvimento da Formação ASP.NET Core do Next Wave Education.
Api focada em gerenciar projetos para clientes e freelancers.

Cada branch representa um módulo da formação seguindo a ordem abaixo:

- **main** -> Implementação de Rest Api com ASP.Net Core
- **feature/implement-entity-framework-core** -> Implementação do acesso a dados utilizando Entity Framework Core
- **feature/implement-clean-architecture** -> Implementação da arquitetura limpa
- **feature/implement-pattern-cqrs** -> Implementação do padrão CQRS
- **feature/implement-pattern-repository** -> Implementação do padrão Repository

Cada branch foi criada a partir da anterior
